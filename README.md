# ChekListBible
Checklist interactive pour suivre sa lecture de la Bible (HTML/CSS/JS)

Application web simple permettant de suivre sa lecture de la Bible en cochant les livres lus.

## 🎯 Objectif
Créer une interface simple et interactive avec HTML, CSS et JavaScript.

## ⚙️ Fonctionnalités
- Liste complète des livres
- Cases à cocher
- Sauvegarde automatique (localStorage)
- Compteur de progression
- Bouton de réinitialisation

## 🛠️ Technologies
- HTML
- CSS
- JavaScript
